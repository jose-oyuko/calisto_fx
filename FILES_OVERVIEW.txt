FILE DESCRIPTIONS
================================================================================

CORE PYTHON FILES
-----------------

1. main.py (Entry Point)
   - TradingBot orchestrator class
   - Startup sequence (Telegram + MT5 login)
   - Message processing pipeline
   - Signal handling (new, modify, close)
   - REPL interface with 10 commands
   - Threading for async Telegram listening
   Usage: python main.py

2. trade_manager.py
   - Trade dataclass (Pydantic-style)
   - TradeManager class
   - JSON persistence
   - Active trade tracking
   - Statistics calculation
   - Context building for LLM
   - Modification history

3. mt5.py
   - MT5Client class
   - Connection management
   - Market order execution
   - Position modification (SL/TP)
   - Order closing (full/partial)
   - Symbol info retrieval
   - Account info retrieval
   Note: Requires MetaTrader 5 terminal on Windows

4. llm.py
   - LLMInterpreter class
   - Pydantic schemas (NewSignal, ModifySignal, CloseSignal, NoSignal)
   - Anthropic Claude integration
   - Tool-based structured output
   - Context-aware interpretation
   - Confidence scoring

5. telegram.py
   - TelegramListener (async)
   - TelegramClient (sync wrapper)
   - Message callback system
   - Dialog/group listing
   - 2FA authentication
   - Real-time message handling

6. utils.py
   - Config class (YAML + .env loader)
   - Logging setup
   - Price parsing (multiple formats)
   - Lot size validation
   - Symbol normalization
   - Risk-reward calculation
   - Terminal color output

================================================================================
REPL COMMANDS REFERENCE
================================================================================

status      Show all active trades
balance     Display MT5 account balance and equity
positions   Show current open positions in MT5
trades      Show recent trade history (last 10)
close <id>  Manually close a trade by ticket or trade_id
stats       Show trading statistics (P&L, win rate, etc.)
pause       Pause message processing (keep bot running)
resume      Resume message processing
help        Show all available commands
exit/quit   Graceful shutdown (saves trades, disconnects)

================================================================================
NEXT STEPS TO GET STARTED
================================================================================

1. Download all files to your computer
2. Install dependencies: pip install -r requirements.txt
3. Setup credentials in .env file
4. Configure settings in config.yaml (optional)
5. Open MetaTrader 5 terminal and login
6. Run: python main.py
7. Follow prompts to login to Telegram
8. Select group to monitor
9. Test with demo signals

For detailed instructions, see QUICKSTART.md

================================================================================
SAFETY REMINDERS
================================================================================

⚠ ALWAYS test on demo account first (minimum 1 week)
⚠ Start with minimum lot sizes (0.01)
⚠ Monitor actively for first few days
⚠ Review all trades in logs
⚠ Have manual override ready (MT5 terminal)
⚠ Set conservative risk limits
⚠ Never risk more than you can afford to lose

Trading carries significant risk. Use at your own discretion.
